# �周易卦爻速查

> 基于蓍草大衍之数的模拟排卦工具，自动解析卦象与爻词

## 功能特性

- 🔮 **蓍草模拟** — 还原大衍之数五十，三变一爻的古老起卦方法
- 📖 **自动解卦** — 根据六爻生成卦象，匹配六十四卦卦义
- ⚡ **爻词解析** — 列出动爻爻辞，分析吉凶悔吝
- 🔄 **变卦计算** — 老阳变阴、老阴变阳，显示本卦与变卦对应关系
- 📊 **可视化展示** — 清晰的卦象符号与爻位标注

## 快速开始

```bash
# 直接运行起卦
python hexagram.py

# 指定爻值（可选，用于复现特定卦）
python hexagram.py --lines 7 8 8 8 6 8
```

## 蓍草起卦原理

> 大衍之数五十，其用四十有九。分而为二以象两，挂一以象三，揲之以四以象四时，归奇于扐以象闰，五岁再闰，故再扐而后挂。

1. **三变**：每次从49根蓍草中随机分配，三变后得36/32/28/24
2. **定爻**：对应老阳(9)/少阴(8)/少阳(7)/老阴(6)
3. **六爻**：重复三次得六爻，从下往上排列
4. **成卦**：下三爻为内卦（贞），上三爻为外卦（悔）
